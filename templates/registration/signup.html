Please complete and submit a W-9 form form in order to receive payments. This can be satisfied via any of the options listed below:

  • Deliver the W-9 form in person to Suite 224 in Guggenheim Hall
  • If you have an existing Multipass login, 'Request Service' and then 'Activate a Current Student or Employee as a Vendor' directly through this link
  • Submit the W-9 by email, omitting the social security number or tax ID number portion from the form by sending it to jslander@mines.edu.

Note: If you are not already affiliated with Mines, once this process has been completed, you will recieve a Campus Wide ID (CWID) with which you can use to register to be a participant on the Mines Crowdsourcing System!
